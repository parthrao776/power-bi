## üöÄ How to Use

1.  **Download:** Ensure you have the `DATA ANALYSIS DASHBOARD.pbix` file on your local machine.
2.  **Open:** Double-click the file. It will automatically open in Power BI Desktop.
3.  **Interact:**
    * Click on charts, graphs, and slicers to filter the data.
    * Use the tabs at the bottom to navigate between different report pages.
    * Right-click on data points to drill down for more detail.
